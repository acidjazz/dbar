dbar
A progress bar inspired by playing too much Destiny
dbar.i() - initiate the progress bar
dbar.perc(0.25) - set the progress to 25%
dbar.perc(0.5) - set the progress to 50%
dbar.perc(0.75) - set the progress to 75%
dbar.d() - complete and remove the progress bar
dbar.set({barColor: #ee5655) - set the bar color
dbar.set({barColor: #76D734) - set the bar color
dbar.set({gradientColor: #76D734) - set the gradient color
